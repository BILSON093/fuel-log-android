# 应用市场文案

## 应用名称

车耗记

## 副标题

油耗电耗与用车费用记录

## 一句话简介

记录加油、充电、保养和费用，自动统计油耗、电耗和真实用车成本。

## 详细介绍

车耗记是一款离线优先的车辆能耗与费用记录工具，适合油车、电车和混动车用户长期记录用车数据。

你可以记录每次加油或充电的数据，包括里程、加油量、充电量、金额、油价、电价、站点和备注。应用会自动计算区间油耗或电耗，并通过图表展示能耗趋势、价格变化、月度费用和站点费用占比。

除了能耗记录，车耗记还支持保养、停车、过路、洗车、维修等用车费用记录，帮助你看到真实每公里成本。

车耗记无需登录，数据默认保存在本机。你可以主动导出 JSON 备份，也可以导出 CSV 表格进行归档或用 Excel 查看。

## 核心功能

- 支持油车、电车独立记录
- 加油记录、充电记录、保养记录、费用记录
- 自动计算油耗、电耗、每公里成本
- 月度费用、价格趋势、站点费用占比图表
- 多车辆管理
- 本地提醒
- JSON 备份导入导出
- CSV 表格导出
- 无需登录，本地存储
