车耗记隐私政策

更新日期：2026年5月24日

一、我们如何处理数据

车耗记是一款本地优先的车辆记录工具。你输入的车辆、里程、加油、充电、保养、费用和备注数据默认仅保存在你的设备本地数据库中。

二、我们收集的信息

当前版本不要求注册或登录，不主动收集账号、手机号、定位、通讯录等个人信息。

你主动输入的数据包括车辆名称、车牌号、里程、费用、站点名称和备注等，这些数据仅用于应用内统计和展示。

三、权限说明

通知权限：用于本地保养、年检、保险等提醒。

文件选择能力：用于你主动导入或导出 JSON、CSV 文件。

四、数据导入导出

你可以主动导出 JSON 备份或 CSV 表格。导出的文件由你自行选择保存位置和分享对象。

导入备份时，应用会读取你选择的 JSON 文件并恢复到本地数据库。

五、第三方 SDK

当前版本未集成广告、统计、云同步或其他第三方 SDK。

如后续版本接入广告或统计服务，我们会在隐私政策中说明 SDK 名称、用途和数据处理方式，并在首次使用前征得必要授权。

六、联系我们

如你对隐私政策或数据处理有疑问，请通过应用市场开发者联系方式与我们联系。
